## 2024-05-20 - Memoize mapped components to prevent expensive list re-renders | Learning: Mapping over arrays and returning React components synchronously on the main thread during rapid state changes (like typing in an uncontrolled/partially-controlled search input) can cause significant UI stuttering, even if the array contents haven't changed. | Action: Wrapped the `.map()` result in a `useMemo` with the array as the dependency, preventing the map loop from executing on every keystroke.

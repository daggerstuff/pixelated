## 2026-03-12 - Prevent NoSQL Injection in Projects API | Vulnerability: NoSQL Injection via Express Query Parser | Learning: Casting `req.query` to string (e.g., `category as string`) satisfies TypeScript but does not prevent Express from passing objects like `{"$ne": "foo"}`, enabling MongoDB NoSQL injection. | Prevention: Always validate runtime types (e.g., `typeof param === 'string'`) or use dedicated parsing/sanitization utilities before querying the database.
